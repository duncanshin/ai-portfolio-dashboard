# AI Portfolio System — Dashboard

Live performance dashboard for the AI Portfolio Management System.

**3-tier risk profiles** | Backtested 2018–2025 | Paper trading via Alpaca

Built by Duncan Shin

## Stack
- React 18 + Vite
- Tailwind CSS
- Recharts
- Deployed on Vercel

## Local Development
```bash
npm install
npm run dev
```

## Deploy
Connected to Vercel via GitHub — auto-deploys on push to `main`.
